- Railway는 최신 커밋을 순차적으로 따라오고 있으나, 현재는 `dc2fd66`까지 반영되어 있고 `2f1c5a3`은 아직 공개 URL에 반영되지 않았다.

## 남은 차단 조건

- 운영 DB mutation workflow 검증을 완료하려면 `VERIFY_DB_WORKFLOW_ACCESS_CODE` 값이 필요하다.
- 이 검증은 테스트 거래, 분개, 리포트, 마감 데이터를 만들고 정리하므로 운영 DB에서 실행하려면 의도적으로 허용해야 한다.
- 실행 형식:

```bash
VERIFY_DB_WORKFLOW_ALLOW_PRODUCTION=1 VERIFY_DB_WORKFLOW_ACCESS_CODE=접근코드 VERIFY_DB_WORKFLOW_BASE_URL=https://honzang-production.up.railway.app npm run verify:db-workflow
```
