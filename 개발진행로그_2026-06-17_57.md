# 개발진행로그 57 - PWA 앱 설치 버튼

## 진행 내용

- 상단 바에 `앱 설치` 버튼을 추가했다.
- 브라우저가 `beforeinstallprompt` 이벤트를 제공하면 기본 PWA 설치 프롬프트를 호출한다.
- 앱 설치가 완료되거나 standalone 모드로 실행 중이면 버튼을 `앱 모드` 상태로 표시한다.
- `verify:pwa` 검증에 PWA 설치 이벤트 처리와 UI 액션 존재 여부를 추가했다.
- README의 웹/앱 지원 설명에 상단 설치 버튼 동작을 반영했다.

## 검증

- `npm run typecheck` 통과
- `npm run lint` 통과
- `npm run build` 통과
- `npm run verify:pwa` 통과
- `npm run smoke:prod` 통과
- Browser 데스크톱 검증
  - 상단 바 `앱 설치` 버튼 렌더링 확인
  - 페이지 전체 가로 오버플로우 없음
  - 콘솔 오류 없음
- Browser 모바일 폭 390px 검증
  - 상단 바 `앱 설치` 버튼 렌더링 확인
  - 페이지 전체 가로 오버플로우 없음
  - 콘솔 오류 없음

## 참고

- iOS Safari처럼 설치 프롬프트 이벤트를 제공하지 않는 브라우저에서는 버튼 클릭 시 브라우저 메뉴의 설치/홈 화면 추가 흐름으로 안내한다.
