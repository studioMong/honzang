# 개발 진행 로그 27

## 목표

- 승인된 자동분개를 기반으로 신고 전 검토용 재무제표 초안을 리포트 화면에 추가한다.
- 저장 리포트 스냅샷에도 당시 재무제표 초안 행을 포함해 월별/기간별 상태를 다시 확인할 수 있게 한다.

## 작업 내용

- 리포트 화면에 `재무제표 초안` 패널을 추가했다.
  - 승인 분개 기준 계정별 최종 잔액을 자산, 부채, 자본, 수익, 비용으로 구분한다.
  - 자산, 부채, 자본(손익 반영), 손익 요약 KPI를 표시한다.
  - 재무제표 초안 CSV 다운로드 버튼을 추가했다.
- 법인세 결산 체크와 신고 패키지에 재무제표 초안 준비 상태를 추가했다.
- 리포트 스냅샷 저장 payload에 `financialStatementRows`를 추가했다.
- 저장된 리포트 상세 화면에서 당시 재무제표 초안 행을 확인할 수 있게 했다.
- README MVP 범위에 승인 분개 기준 원장과 재무제표 초안을 반영했다.

## 검증

- `npm run typecheck` 통과
- `npm run lint` 통과
- `npm run verify:samples` 통과
- `npm run build` 통과
- `npm run smoke:prod` 통과
- 브라우저 확인
  - `http://localhost:3000/?view=reports`에서 `재무제표 초안` 패널 렌더링 확인
  - 데스크톱 기본 폭에서 CSV 버튼과 빈 상태 안내 확인
  - 모바일 390px 폭에서 페이지 전체 가로 오버플로우 없음 확인

## 남은 확인

- 현재 샘플 데이터는 승인 분개가 없어 재무제표 초안이 빈 상태로 표시된다.
- 실제 CSV 업로드 후 자동분개 승인 흐름에서 재무제표 행이 생성되는 실사용 검증이 다음 우선순위다.
