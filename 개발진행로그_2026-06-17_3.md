# 혼자장부 개발 진행 로그

작성일: 2026-06-17

## 진행한 작업

- `증빙함` 탭 추가
  - 증빙 유형 선택
  - 발행일, 거래처, 사업자등록번호 입력
  - 공급가액, 부가세, 합계 입력
  - 거래 매칭
  - 파일명 기록
  - 증빙 목록 조회

- 증빙 API 추가
  - `GET /api/evidences`
  - `POST /api/evidences`
  - Railway Postgres 연결 시 `Evidence` 테이블에 저장
  - 거래와 매칭하면 해당 거래의 `evidenceStatus`를 `MATCHED`로 갱신
  - DB가 없으면 샘플 데이터 모드로 동작

- `자동분개` 탭 추가
  - 거래별 분개 초안 생성
  - 입금 매출 분개
  - 비용 지출 분개
  - 매입 부가세대급금 후보
  - 대표자/외주/해외 SaaS 등 검토 경고 표시
  - 차변/대변 균형 여부 표시

- URL 접근 확장
  - `/?view=evidences`
  - `/?view=journals`

## 검증

통과한 명령:

```bash
npm run lint
npm run typecheck
npm run build
```

화면/API 확인:

```bash
curl -s 'http://localhost:3000/?view=evidences'
curl -s 'http://localhost:3000/?view=journals'
curl -s 'http://localhost:3000/api/evidences'
```
