# 개발 진행 로그 85

## 작업 단위
- 증빙 금액 불일치 리뷰를 대시보드와 신고 준비 리포트 요약에 노출했다.

## 변경 내용
- `src/components/app-workspace.tsx`
  - 리포트 패널이 서버/상태 기반 `reviewItems`를 받아 기간별 열린 리뷰를 사용하도록 변경했다.
  - 대시보드 검토 KPI에 증빙 금액 불일치 건수를 함께 표시한다.
  - 오늘 할 일의 검토함 카드가 증빙 금액 불일치를 포함하면 더 높은 우선순위로 보이게 했다.
  - 신고 준비 리포트 KPI에 `증빙 불일치` 건수를 추가했다.
  - 신고 준비 체크리스트에 `증빙 금액` 항목을 추가했다.
  - 저장 리포트 payload와 마감 payload에 리뷰 CSV를 포함해 저장된 리포트 상세에서도 불일치 건수를 볼 수 있게 했다.
  - ZIP/XLSX 신고 패키지 요약에 `증빙 금액 불일치` 항목을 추가했다.

## 검증
- `npm run typecheck` 통과
- `npm run lint` 통과
- `git diff --check` 통과
- `npm run verify:mvp` 통과
- 브라우저 확인
  - 대시보드 KPI: `검토 5건 · 위험 2건 · 불일치 0건` 렌더링 확인
  - 리포트 KPI: `증빙 불일치 0건` 렌더링 확인
  - 리포트 체크리스트: `증빙 금액 0건 불일치` 렌더링 확인
