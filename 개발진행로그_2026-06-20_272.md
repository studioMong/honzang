## 변경 사항

- `scripts/verify-railway-access.mjs` 추가.
  - `/api/auth/session`이 접근 보호 활성 상태를 보고하는지 확인한다.
  - 익명 사용자가 인증되지 않은 상태인지 확인한다.
  - `/?view=reports`가 `/access`로 리다이렉트되고 `next` 경로가 보존되는지 확인한다.
  - 익명 `/api/transactions` 요청이 HTTP 401과 `AUTH_REQUIRED`로 차단되는지 확인한다.
  - `/api/version`, `/api/health`, `/manifest.webmanifest`, 샘플 CSV는 공개 리소스로 계속 응답하는지 확인한다.
- `package.json`
  - `verify:railway-access` 스크립트 추가.
- `README.md`, `docs/railway-cutover.md`
  - Railway 공개 URL 확인 절차에 `npm run verify:railway-access`를 추가했다.
- `scripts/verify-deployment-config.mjs`
  - 새 스크립트 존재와 핵심 검증 문구를 확인하도록 추가했다.
