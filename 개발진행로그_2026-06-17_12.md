# 개발진행로그 12

## 목표

초기 사용자가 1인법인 신고 준비에 필요한 기본정보를 한눈에 확인할 수 있게 한다.

## 변경 내용

- 대시보드에 `1인법인 신고 준비` 패널을 추가했다.
- 설정 화면에 `필수정보 상태` 패널을 추가했다.
- 법인 기본정보, 부가세 신고 기준, 법인세 결산월, 원천세 대상, 매출 과금 방식을 공통 체크리스트로 계산한다.
- 사업자등록번호, 업종 등 기본정보가 비어 있으면 입력 필요 상태로 표시한다.
- 대표자 급여, 직원 급여, 외주 지급이 켜져 있으면 원천세 관리 필요 상태로 표시한다.

## 검증 결과

- `npm run lint` 통과
- `npm run typecheck` 통과
- `npm run verify:samples` 통과
- `npm run build` 통과
- 로컬 dev 서버에서 `/`와 `/?view=settings`가 HTTP 200 응답하는 것을 확인했다.

## 제한 사항

- Playwright 번들 브라우저가 설치되어 있지 않고 시스템 Chrome도 현재 REPL 샌드박스에서 실행되지 않아 시각적 스크린샷 검증은 수행하지 못했다.
