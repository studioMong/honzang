# 개발진행로그 25

## 목표

Railway에서 실행될 Next standalone 서버 경로를 로컬에서 반복 검증할 수 있게 한다.

## 변경 내용

- `scripts/smoke-production.mjs`를 추가했다.
- `npm run smoke:prod` 스크립트를 추가했다.
- 스모크 테스트는 `npm run build` 후 `npm run start`로 standalone 서버를 띄운다.
- `/api/version`, `/api/health`, `/api/reviews`, `/` 응답을 확인한다.
- README 로컬 실행 섹션에 프로덕션 스모크 검증 명령을 추가했다.

## 검증 결과

- `npm run build` 통과
- `npm run smoke:prod` 통과
- `npm run typecheck` 통과
- `npm run lint` 통과
- `npm run verify:samples` 통과
