# 개발 진행 로그 125

## 작업 단위
- 마감 잠금된 기간의 검토 항목 상태 변경을 차단하도록 리뷰 API를 보강했다.

## 변경 파일
- `src/app/api/reviews/route.ts`
- `scripts/verify-db-workflow.ts`

## 구현 내용
- `PATCH /api/reviews`에서 검토 항목의 연결 거래일을 조회한다.
- 연결 거래일이 마감 잠금된 기간에 포함되면 HTTP 409와 `PERIOD_CLOSED`를 반환한다.
- 거래에 연결되지 않은 검토 항목은 기존처럼 상태 변경을 허용한다.
- DB 워크플로우 검증에 마감 후 증빙 금액 불일치 검토 항목을 `RESOLVED`로 바꾸려는 시나리오를 추가했다.

## 실행 명령
- `npm run typecheck`
- `npm run lint`
- `git diff --check`
- `npm run verify:mvp`
- `if [ -n "$DATABASE_URL" ]; then printf 'DATABASE_URL=set\n'; else printf 'DATABASE_URL=missing\n'; fi`
