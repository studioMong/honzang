# 개발 진행 로그 216

## 작업 주제
- 운영 준비 점검에 Railway Postgres 스키마 상태 확인을 추가했다.

## 구현 내용
- `/api/operations/readiness`에 `databaseSchema` 점검을 추가했다.
- 필수 Prisma 테이블 목록을 명시하고 `information_schema.tables`에서 public schema 테이블 존재 여부를 확인한다.
- `DATABASE_URL`이 없으면 중복 blocker를 만들지 않고 `blue` 대기 상태로 표시한다.
- DB가 연결되어 있는데 필수 테이블이 빠져 있으면 `red` 상태와 누락 테이블 목록을 반환한다.
- 모든 필수 테이블이 있으면 `green` 상태로 필수 테이블 개수를 표시한다.

## 검증 보강
- `scripts/verify-operations-readiness.mjs`에서 `databaseSchema` 체크의 순서, 상태, 안내 문구를 검증하도록 갱신했다.
- `scripts/verify-deployment-config.mjs`에서 readiness 라우트가 `databaseSchemaCheck`, `REQUIRED_DATABASE_TABLES`, `information_schema.tables` 계약을 포함하는지 확인하도록 했다.
