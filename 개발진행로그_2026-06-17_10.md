# 개발 진행 로그 - 2026-06-17 10차

## 이번 목표
- 사용자가 어떤 CSV를 준비해야 하는지 앱 안에서 바로 확인할 수 있게 만든다.
- 통장, 카드, 홈택스, PG 샘플 CSV를 제공해 업로드 테스트를 쉽게 한다.

## 구현 내용
- `public/samples/` 아래에 샘플 CSV를 추가했다.
  - `bank-transactions.csv`
  - `card-transactions.csv`
  - `hometax-sales.csv`
  - `hometax-purchases.csv`
  - `pg-settlements.csv`
- 업로드 화면에 현재 선택한 자료 유형의 샘플 다운로드 버튼을 추가했다.
- 업로드 화면 파일 선택 영역 아래에 자료 유형별 샘플 링크를 추가했다.
- README에 샘플 CSV 위치와 사용 방식을 기록했다.

## 검증
- `npm run lint` 통과.
- `npm run typecheck` 통과.
- `npm run build` 통과.
- 로컬 파일 서빙 확인:
  - `GET /samples/bank-transactions.csv` 본문 확인.
  - `GET /samples/card-transactions.csv`가 `text/csv`로 응답하는 것 확인.
- 로컬 브라우저 확인:
  - `/?view=imports`에서 `통장 샘플` 버튼 노출 확인.
  - 통장, 카드, 홈택스 매출, 홈택스 매입, 현금영수증, PG 정산 샘플 링크 노출 확인.

## 다음 우선순위
