# 개발 진행 로그 81

## 작업 단위
- 거래 총액과 연결 증빙 합계가 맞지 않는 경우를 신고 전 검토 항목으로 자동 생성하도록 개선했다.

## 변경 내용
- `src/lib/server/evidence-amount-reviews.ts`
  - 거래 총액과 연결 증빙 합계를 비교하는 공통 리뷰 빌더를 추가했다.
  - 1원 이하 반올림 차이는 허용하고, 큰 차이는 `DANGER`, 작은 차이는 `WARNING`으로 분류한다.
  - 권고 문구에 거래금액, 연결 증빙 합계, 차액을 포함한다.
- `src/app/api/reviews/route.ts`
  - 리뷰 후보 계산 시 거래와 연결 증빙을 함께 조회한다.
  - 기존 거래 기반 리뷰에 증빙 금액 불일치 리뷰를 추가한다.
  - 기존 저장 리뷰의 상태는 유지하고, 활성 후보의 심각도/권고 문구만 최신 계산값으로 보정한다.
- `scripts/verify-sample-csvs.ts`
  - 증빙 합계가 거래 총액보다 30,000원 부족한 케이스가 `DANGER` 리뷰를 생성하는지 검증한다.
  - 증빙 합계가 거래 총액과 일치하면 리뷰가 생성되지 않는지 검증한다.
