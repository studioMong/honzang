# 개발 진행 로그 - 2026-06-17 6차

## 이번 목표
- 신고 준비 리포트를 실제 마감 업무에 더 가깝게 만든다.
- 부가세, 원천세, 법인세 준비 자료를 앱에서 직접 확인하고 CSV로 내보낼 수 있게 한다.

## 구현 내용
- 리포트 화면에 `신고 패키지` 섹션을 추가했다.
  - 거래 분류, 증빙, 부가세, 원천세, 자동분개, 법인세 준비 상태를 한 표에서 확인한다.
  - `honzang-filing-package.csv` 다운로드를 추가했다.
- `원천세 후보` 표를 추가했다.
  - 급여, 외주비, 프리랜서, 사업소득/기타소득 후보 거래를 자동 추출한다.
  - 사업소득/기타소득 후보는 3.3% 예상 원천세를 표시한다.
  - `honzang-withholding-candidates.csv` 다운로드를 추가했다.
- `법인세 결산 체크` 표를 추가했다.
  - 매출, 비용, 손익, 증빙 누락, 대표자 거래, 미분류 거래, 승인 분개, 계정별 원장 상태를 확인한다.
  - `honzang-corporate-tax-prep.csv` 다운로드를 추가했다.
- 손익/부가세 집계 로직을 계정 유형 기준으로 보정했다.
  - 대표자차입금 같은 부채성 입금은 매출에서 제외한다.
  - 비용 계정이 아닌 출금은 비용에서 제외한다.
  - 해외 SaaS처럼 부가세 금액이 명시되지 않은 거래는 매입세액으로 자동 확정하지 않는다.

## 검증
- `npm run lint` 통과.
- `npm run typecheck` 통과.
- `npm run build` 통과.
- 로컬 브라우저 확인:
  - `/?view=reports`에서 `신고 패키지`, `원천세 후보`, `법인세 결산 체크` 섹션 노출 확인.
  - 샘플 대표자 입금이 매출에서 제외되어 매출 공급가액이 `1,000,000원`으로 표시되는 것을 확인.
  - 해외 SaaS 부가세가 매입세액에서 제외되어 예상 부가세 납부액이 `60,000원`으로 표시되는 것을 확인.
