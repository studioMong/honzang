# 개발 진행 로그 105

## 작업 단위
- 신고 리포트 스냅샷의 payload 기간이 요청한 리포트 기간과 어긋나도 저장될 수 있는 문제를 서버에서 차단했다.

## 변경 내용
- `src/app/api/reports/route.ts`
  - `calculatedPayload.period`가 `YYYY-MM`이면 해당 월의 시작일/종료일이 요청 `periodStart`/`periodEnd`와 일치하는지 검증한다.
  - `calculatedPayload.period`가 객체이고 `start`/`end`가 있으면 요청 기간과 일치하는지 검증한다.
  - 불일치 시 `REPORT_PAYLOAD_PERIOD_MISMATCH` 400 응답과 상세 issues를 반환한다.
- `scripts/smoke-production.mjs`
  - 요청 기간은 2026-06인데 payload 기간이 2026-05인 리포트 저장 요청이 거절되는지 검증한다.

## 실행 명령
- `npm run typecheck`
- `npm run lint`
- `git diff --check`
- `npm run verify:mvp`

## 검증 결과
- TypeScript typecheck 통과.
