# 개발진행로그 20

## 목표

사용자가 어떤 CSV 자료를 이미 가져왔는지 확인할 수 있게 해 업로드 상태와 중복 방지 결과를 투명하게 보여준다.

## 변경 내용

- `AppImportBatch` 타입을 추가했다.
- `ImportBatch` 직렬화 함수를 추가했다.
- `/api/imports`에 `GET`을 추가해 최근 50개 업로드 이력을 반환한다.
- CSV import `POST` 응답에 `importBatch` 객체를 포함했다.
- 앱 초기 refresh에서 업로드 이력을 함께 조회한다.
- 업로드 화면에 `최근 업로드` 표를 추가했다.
- CSV import 후 새 import batch를 화면 이력에 병합한다.

## 표시 항목

- 가져온 시간
- 자료 유형
- 원본 파일명
- 행 수
- 파일 해시 앞 12자리
