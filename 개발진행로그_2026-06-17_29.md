# 개발 진행 로그 29

## 목표

- 신고 준비자료를 한 파일로 보관하거나 전달할 수 있도록 리포트 패키지 다운로드를 추가한다.

## 작업 내용

- 리포트 화면에 신고 패키지 JSON 다운로드를 추가했다.
  - 신고 준비 영역에는 `패키지 JSON` 버튼을 추가했다.
  - 신고 패키지 섹션에는 `JSON` 버튼을 추가했다.
- JSON 패키지에 다음 데이터를 포함하도록 구성했다.
  - 회사 기본 정보
  - 리포트 기간
  - 요약 손익/부가세/검토 수치
  - 신고 일정
  - 신고 패키지 체크리스트
  - 거래 목록
  - 부가세 리포트
  - 검토 항목
  - 원천세 후보
  - 법인세 준비 체크
  - 재무제표 초안
  - 계정별 원장
- README MVP 범위에 신고 준비자료 JSON 패키지 다운로드를 반영했다.

## 검증

- `npm run typecheck` 통과
- `npm run lint` 통과
- `npm run verify:samples` 통과
- `npm run build` 통과
- `npm run smoke:prod` 통과
- 브라우저 확인
  - 리포트 화면에서 `패키지 JSON` 버튼 렌더링 확인
  - 신고 패키지 섹션에서 `JSON` 버튼 렌더링 확인
  - 모바일 390px 폭에서 신고 패키지 영역의 페이지 전체 가로 오버플로우 없음 확인
