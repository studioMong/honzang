# 개발 진행 로그 32

## 목표

- 저장된 리포트 스냅샷을 앱에서 직접 정리할 수 있게 한다.

## 작업 내용

- 저장된 리포트 목록에 `삭제` 버튼을 추가했다.
- 리포트 상세 화면에도 `삭제` 버튼을 추가했다.
- 삭제 중인 리포트 버튼은 `삭제 중` 상태로 비활성화한다.
- 삭제 성공 시 목록에서 해당 리포트를 제거하고, 상세 화면에서 삭제한 경우 상세 패널을 닫는다.

## 검증

- `npm run typecheck` 통과
- `npm run lint` 통과
- `npm run verify:samples` 통과
- `npm run build` 통과
- `npm run smoke:prod` 통과
- 브라우저 확인
  - 리포트 화면에서 `스냅샷 저장` 실행 후 `저장된 리포트` 섹션과 `열기`, `삭제` 버튼 표시 확인
  - `삭제` 실행 후 저장된 리포트 섹션이 사라지는 것 확인

## 비고

- 삭제 API는 이전 작업에서 추가한 `/api/reports DELETE`를 사용한다.
- 현재 삭제는 즉시 실행된다. 이후 실제 사용자 데이터 보호를 위해 확인 모달을 추가하는 것이 좋다.
