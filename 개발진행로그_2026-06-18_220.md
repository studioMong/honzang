## 작업 주제

## 구현 내용
- `scripts/verify-database-schema-contract.mjs`를 추가했다.
- `prisma/schema.prisma`의 `model` 목록과 `src/lib/server/database-schema.ts`의 `REQUIRED_DATABASE_TABLES`가 순서까지 동일한지 검증한다.
- DB 스키마 헬퍼가 `information_schema.tables`와 public schema를 조회하는지도 확인한다.
- `npm run verify:database-schema-contract`를 추가하고 `npm run verify:mvp`에 연결했다.
- `scripts/verify-deployment-config.mjs`에서도 새 verifier 존재와 package script 계약을 확인한다.
