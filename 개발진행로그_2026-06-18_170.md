## 구현 내용
- `src/lib/server/request-json.ts`를 추가해 JSON 요청 본문을 안전하게 읽고, 크기 제한과 JSON 문법 오류를 공통 처리하게 했다.
- `src/app/api/reports/route.ts`의 리포트 저장/삭제 요청에 공통 JSON 파서를 적용했다.
- `src/app/api/closing-periods/route.ts`의 마감 잠금/해제 요청에 공통 JSON 파서를 적용했다.
- 잘못된 JSON은 `INVALID_JSON_PAYLOAD`와 HTTP 400으로 응답한다.
- 기본 크기 제한을 넘는 JSON 요청은 `JSON_PAYLOAD_TOO_LARGE`와 HTTP 413으로 응답한다.
- `scripts/smoke-production.mjs`에 리포트/마감 API의 malformed JSON 거부 검증을 추가했다.
