# 개발 진행 로그 139

## 작업 단위
- 자동분개 승인/취소 UI의 실패 피드백을 보강했다.

## 변경 파일
- `src/components/app-workspace.tsx`

## 구현 내용
- `JournalDraftsPanel`에 `journalMessage` 상태를 추가했다.
- 단건 분개 승인 실패와 네트워크 오류를 배너 메시지로 표시한다.
- 단건 분개 승인 취소 실패와 네트워크 오류를 배너 메시지로 표시한다.
- 일괄 승인 중 실패하면 몇 건 승인 후 중단됐는지 표시한다.
- 단건 승인, 승인 취소, 일괄 승인 성공 시 성공 메시지를 표시한다.

## 실행 명령
- `npm run typecheck`
- `npm run lint`
- `git diff --check`
- `npm run verify:mvp`
- `if [ -n "$DATABASE_URL" ]; then printf 'DATABASE_URL=set\n'; else printf 'DATABASE_URL=missing\n'; fi`
