# 개발 진행 로그 143

## 작업 단위
- 설정성 데이터 API의 데이터 변경과 감사 로그 기록을 원자 작업으로 묶었다.

## 변경 파일
- `src/app/api/companies/route.ts`
- `src/app/api/vendors/route.ts`
- `src/app/api/classification-rules/route.ts`
- `src/app/api/csv-templates/route.ts`
- `scripts/verify-db-workflow.ts`

## 구현 내용
- 회사 설정 저장과 `COMPANY_UPDATE` 감사 로그 기록을 하나의 Prisma 트랜잭션으로 처리한다.
- 거래처 기본값 생성/수정/삭제와 `VENDOR_*` 감사 로그 기록을 하나의 Prisma 트랜잭션으로 처리한다.
- 자동분류 규칙 생성/수정/삭제와 `CLASSIFICATION_RULE_*` 감사 로그 기록을 하나의 Prisma 트랜잭션으로 처리한다.
- CSV 매핑 템플릿 삭제와 `CSV_TEMPLATE_DELETE` 감사 로그 기록을 하나의 Prisma 트랜잭션으로 처리한다.
- DB 워크플로우 검증에 거래처 기본값 생성, 수정, 없는 거래처 삭제 실패 시나리오를 추가했다.

## 실행 명령
