## 운영 상태
- GitHub `origin/main`은 최신 커밋을 가리키고 있었다.
- Railway 공개 URL은 앱, DB, 접근 보호는 정상이나 배포 커밋이 직전 커밋에 머물러 있었다.
- 이 상태는 코드 변경보다 Railway Deployments/Public Networking/GitHub 연결 상태 확인이 필요한 운영 조건이다.
