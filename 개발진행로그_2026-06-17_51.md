# 개발진행로그 51 - 부가세 신고 준비표 보강

## 진행 내용

- 리포트의 `부가세` 다운로드 자료를 단순 합계표에서 신고 입력 전 정리표로 확장했다.
- 부가세 CSV, 신고 패키지 ZIP, 신고 패키지 XLSX의 `vat-report`에 다음 행을 포함한다.
  - 과세 매출 공급가액과 매출세액 요약
  - 매입세액 공제 추정 요약
  - 증빙 미확인으로 공제 보류할 후보
  - 해외 SaaS, 불공제 가능 비용 등 검토 후보
  - 매출/매입 거래별 공급가액, 세액, 근거, 검토 메모
- README의 MVP 범위를 `부가세 신고 입력 전 정리표`로 구체화했다.

## 검증

- `npm run typecheck` 통과
- `npm run lint` 통과
- `npm run build` 통과
- `npm run smoke:prod` 통과

## 참고

- 이 자료는 홈택스 입력 전 검토용이며, 최종 신고 전에는 홈택스/국세청 공지와 증빙 원본 확인이 필요하다.
- 커밋 푸시 후 `RAILWAY_EXPECTED_COMMIT=ba159c0 npm run verify:railway`를 실행했지만 공개 URL은 여전히 기존 정적 프로젝트 요약 페이지를 서빙하고 `/api/version`, `/api/health`, `/manifest.webmanifest`는 404로 응답했다.
