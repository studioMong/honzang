# 개발진행로그 81

## 주제

설정 화면 운영 준비 점검 추가

## 변경 내용

- `/api/operations/readiness` 보호 API를 추가했다.
- 운영 준비 API가 Postgres 연결, 접근코드 보호, 접근 쿠키 salt, 공개 앱 URL, 서버 런타임, Railway 메타데이터를 점검한다.
- 민감한 환경변수 값은 노출하지 않고 설정 여부, 상태, 다음 작업만 반환하도록 구성했다.
- 설정 화면에 `운영 준비 점검` 패널을 추가해 배포 직후 확인해야 할 상태를 표로 보여준다.
- smoke 검증이 `/api/operations/readiness` JSON과 설정 화면의 운영 준비 패널 존재를 확인하도록 확장했다.
- 접근코드 보호 검증에서 운영 준비 API가 비로그인 상태에서는 `401`, 로그인 후에는 정상 응답하는지 확인한다.
- README 배포 후 확인 항목에 설정 화면 운영 준비 점검을 추가했다.
