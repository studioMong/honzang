혼자장부 프로젝트 정리

1인법인 대표가 직접 장부를 정리하고, 신고 전 숫자를 검토한 뒤, 홈택스에 직접 신고할 수 있게 돕는 회계/장부 관리 소프트웨어 기획 문서

작성일: 2026-06-15 제품명: 혼자장부 대상: 1인법인, 소규모 법인 범위: 장부 정리 및 신고 준비

1. 한 줄 정의

혼자장부는 세무기장 대행 서비스가 아니라, 대표자가 직접 장부를 정리하고 신고 전 자료를 준비할 수 있게 해주는 셀프 회계 정리 도구다.

초기 제품은 홈택스 자동 신고 제출까지 하지 않는다. 사용자가 통장, 카드, 홈택스 자료를 올리면 앱이 거래를 분류하고 증빙을 매칭하고 부가세, 원천세, 법인세 신고 준비용 숫자를 보여준다. 최종 확인과 신고는 사용자가 직접 한다.

2. 제품 경계

허용 가능한 포지셔닝

  • 1인법인을 위한 셀프 장부 정리 도구
  • 거래내역 자동 분류 및 자동 분개 초안
  • 증빙 누락 알림
  • 부가세, 원천세, 법인세 신고 준비용 집계
  • 사용자가 직접 홈택스에 신고하기 위한 참고자료 제공

피해야 할 포지셔닝

  • 세무기장 대행
  • 법인세 신고 대행
  • 절세 자문
  • 세무사가 필요 없는 완전 자동 신고
  • 고객 대신 홈택스 신고 제출

핵심은 사용자에게 추천, 집계, 검토 필요 항목을 제공하는 것이다. 세무 판단을 확정하거나 신고를 대리하는 구조로 가면 세무대리 이슈가 생길 수 있다.

3. 실제 세무 실무 흐름

1인법인 실무는 대략 거래 발생 → 증빙 확보 → 장부 반영 → 세금별 집계 → 신고 순서로 진행된다.

  1. 법인통장 거래내역을 내려받는다.
  2. 법인카드 사용내역을 내려받는다.
  3. 홈택스에서 전자세금계산서, 계산서, 현금영수증, 신용카드 매입자료를 확인한다.
  4. 매출이 있으면 세금계산서 발행 여부, 카드/PG 정산 여부, 입금 여부를 맞춘다.
  5. 급여, 외주비, 기타소득 지급이 있으면 원천세 대상인지 확인한다.
  6. 거래마다 계정과목을 붙인다.
  7. 증빙 없는 지출, 개인적 사용 의심 지출, 대표자 입출금 같은 위험 거래를 따로 검토한다.
  8. 월별 손익, 부가세 예상액, 원천세 납부액을 확인한다.
  9. 분기, 반기, 연말에 신고용 자료를 만든다.

혼자장부는 이 과정을 앱 안에서 순서대로 처리하게 만든다. 사용자는 매달 자료만 올리고, 앱은 자동 분류와 검토 항목을 보여준다.

5. 첫 버전 MVP

첫 버전의 목표는 자동 신고가 아니라 신고 준비 패키지 생성이다.

필수 기능

  • 회사 설정: 법인명, 사업자번호, 업종, 과세유형, 결산월, 대표자 급여 여부
  • 자료 업로드: 통장 CSV, 카드 CSV, 홈택스 매출/매입자료 CSV, PG 정산 CSV
  • 거래 자동 분류: 매출, 광고비, 지급수수료, 소모품비, 통신비, 복리후생비, 외주비, 급여, 세금과공과 등
  • 증빙 매칭: 통장/카드 거래와 세금계산서, 현금영수증, 카드전표 연결
  • 자동 분개 초안: 차변/대변 분개 생성
  • 검토함: 애매한 거래만 사용자가 확인
  • 부가세 리포트: 매출세액, 매입세액, 불공제 후보, 납부/환급 예상액
  • 원천세 리포트: 급여, 사업소득, 기타소득 지급액과 원천징수세액
  • 법인세 준비 리포트: 손익계산서, 재무상태표 초안, 계정별 원장, 증빙 누락 목록
  • 마감 체크리스트: 이번 달, 이번 분기, 연말 결산 전 확인할 일

8. 자동 분개 예시

법인카드로 네이버 검색광고 110,000원 결제

차변: 광고선전비 100,000
차변: 부가세대급금 10,000
대변: 미지급금 또는 보통예금 110,000

고객사 입금 1,100,000원, 전자세금계산서 있음

차변: 보통예금 1,100,000
대변: 매출 1,000,000
대변: 부가세예수금 100,000

대표가 법인통장에 돈을 넣음

차변: 보통예금
대변: 대표자차입금

대표가 법인 돈을 개인적으로 가져감

차변: 가지급금 또는 대표자대여금 후보
대변: 보통예금

대표자에게 나간 돈, 대표자가 넣은 돈, 법인카드 개인 사용 의심 거래는 1인법인에서 특히 위험하다. 혼자장부는 이 유형을 자동으로 대표자 거래 검토 필요로 올려야 한다.

9. 경고해야 할 항목

  • 증빙 없는 비용
  • 법인카드 개인 사용 의심 거래
  • 대표자에게 나간 돈
  • 대표자가 넣은 돈
  • 접대비, 경조사비, 식대성 지출
  • 해외 SaaS 결제
  • 외주비인데 원천세 신고가 없는 건
  • 세금계산서 매출은 있는데 입금이 안 된 건
  • 입금은 있는데 매출 증빙이 없는 건
  • 부가세 공제 불가 가능성이 있는 건
  • 3월 법인세 신고 전 재무제표가 맞지 않는 건
  • 외부조정 대상 가능성이 있는 건

10. 부가세 계산 흐름

부가세는 큰 틀에서 매출세액에서 매입세액을 빼고, 경감·공제세액과 기납부세액, 가산세 등을 반영해 납부 또는 환급 세액을 계산한다. 국세청의 부가세 세액계산 흐름도는 일반과세자의 매출세액, 매입세액, 납부/환급세액 구조를 안내한다. 국세청 부가가치세 세액계산 흐름도

혼자장부 1차 버전은 복잡한 공제/감면 자동 판단보다 다음 집계를 정확히 만드는 데 집중한다.

  • 세금계산서 매출 공급가액/부가세
  • 카드/현금영수증/PG 매출 공급가액/부가세
  • 세금계산서 매입 공급가액/부가세
  • 카드/현금영수증 매입 공급가액/부가세
  • 매입세액 불공제 후보
  • 증빙 없는 지출
  • 납부 또는 환급 예상액

11. 법인세 자동화 범위

법인세는 단순 매출/비용 합산으로 끝나지 않는다. 국세청은 기업회계상 당기순손익을 기초로 세법상 익금과 손금을 조정하여 과세소득을 계산하는 절차를 세무조정이라고 설명한다. 국세청 법인세 신고절차

따라서 혼자장부의 초기 목표는 다음까지다.

  • 재무상태표 초안
  • 포괄손익계산서 초안
  • 계정별 원장
  • 증빙 누락 목록
  • 대표자 거래 목록
  • 세무조정 검토 필요 항목
  • 법인세 신고 전 체크리스트

법인세 신고서 자동 작성/제출은 후순위다. 특히 외부조정 대상 가능성이 있으면 전문가 검토 필요 경고를 띄운다.

12. 개발 순서

1단계: CSV 기반 로컬/웹 MVP

  1. 회사 설정
  2. 통장 CSV 업로드
  3. 카드 CSV 업로드
  4. 거래 목록 화면
  5. 계정과목 수동 지정
  6. 자동 분류 규칙 저장
  7. 증빙 파일 첨부
  8. 월별 손익 리포트
  9. 부가세 집계
  10. 원천세 체크리스트
  11. 법인세 결산 준비 리포트

2단계: 자동화 강화

  1. 은행/카드 CSV 포맷 템플릿 확장
  2. 홈택스 내려받기 자료 자동 파서
  3. PG사 정산자료 파서
  4. 반복 거래 자동 규칙 추천
  5. 증빙 자동 매칭 고도화
  6. 부가세 신고서 입력값 형태로 출력

3단계: 외부 서비스화

  1. 다중 회사/다중 사용자 구조
  2. 권한 관리
  3. 파일 암호화 및 보안 감사 로그
  4. 상표 확인 및 도메인 확보
  5. 세무사 제휴 검토
  6. 유료 플랜 설계

13. 다음 프로젝트에서 바로 사용할 요구사항

프로젝트명: 혼자장부

7. 앱이 부가세/원천세/법인세 준비 리포트 생성
8. 사용자가 홈택스에서 직접 신고

첫 구현 우선순위:
1. 거래 업로드와 파싱
2. 거래 목록과 계정과목 지정
3. 자동 분류 규칙
4. 증빙 상태 표시
5. 부가세 집계
6. 월별 손익 리포트
7. 신고 준비 체크리스트

14. 보안과 데이터 원칙

  • 원본 CSV와 업로드 파일은 변경하지 않고 별도 보관한다.
  • 분류 결과, 분개 결과, 검토 상태는 별도 테이블로 관리한다.
  • 사업자등록번호, 계좌, 거래처, 급여 정보는 민감정보에 준해 다룬다.
  • 외부 서비스화 전에는 파일 암호화, 접근 로그, 백업/삭제 정책이 필요하다.
  • 홈택스 인증서/스크래핑 연동은 초기 MVP에서 제외한다.
  • 신고 결과와 납부 결과는 사용자가 직접 입력하거나 PDF/접수증을 업로드하는 방식부터 시작한다.

15. 공식 참고 링크

  • 세무사법 제2조 - 세무대리 업무 범위 확인용
  • 세무사법 제20조 - 세무대리 제한 확인용
  • 국세청 부가가치세 신고납부기한
  • 국세청 부가가치세 세액계산 흐름도
  • 국세청 원천세 신고납부기한
  • 국세청 지급명세서 제출
  • 국세청 법인세 신고절차
  • 국세청 전자세금계산서 안내
  • KIPRIS 상표 검색 - 혼자장부 상표 확인 필요

주의: 세법, 신고기한, 전자신고 방식은 바뀔 수 있다. 실제 출시 전에는 최신 국세청 안내와 세무사/법률 전문가 검토가 필요하다.
