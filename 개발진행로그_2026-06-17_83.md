# 개발진행로그 83

## 주제

신고 리포트 복식부기 검증 보강

## 변경 내용

- 신고 준비 리포트에 `복식부기 검증` 패널을 추가했다.
- 승인 분개 기준으로 `차변/대변 합계`, `회계등식`, `계정별 원장`, `재무제표 초안` 4개 검증 행을 표시한다.
- 차변/대변 차액 또는 회계등식 차액이 있으면 신고 준비 차단 상태로 드러나게 했다.
- 대시보드의 최근 월 신고 준비율에도 복식부기 검증 행을 반영했다.
- 신고 패키지 JSON, ZIP, XLSX에 `journalIntegrityRows`, `csv/journal-integrity.csv`, `복식검증` 시트를 추가했다.
- 저장 리포트와 월 마감 스냅샷에도 복식부기 검증 결과를 포함했다.
- smoke 검증이 리포트 화면의 `복식부기 검증` 패널을 확인하도록 확장했다.

## 검증

- `npm run typecheck`: 통과
- `npm run lint`: 통과
- `npm run build`: 통과
- `npm run verify:deployment-config`: 통과
- `npm run smoke:prod`: 통과
- `npm run verify:mvp`: 통과

  - `차변/대변 합계`, `회계등식`, `계정별 원장`, `재무제표 초안` 4개 행 표시
  - 기본 데스크톱 폭에서 가로 오버플로 없음 확인
