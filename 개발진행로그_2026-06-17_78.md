## 판단

현재 공개 URL `https://honzang-production.up.railway.app`은 최신 Next.js standalone 서버가 아니라 이전 정적/legacy 서비스에 연결되어 있다. 로컬 빌드, MVP 검증, 접근제어 검증은 통과했으므로 코드 산출물 문제보다는 Railway 서비스 연결/도메인/배포 설정 문제로 보는 것이 맞다.

## Railway Dashboard에서 확인할 항목

- Public domain `honzang-production.up.railway.app`이 어느 서비스에 붙어 있는지
- 해당 서비스의 최신 Deploy source commit이 `e2c15fdb54dce78710c3c4d792d939a34f2f3e73`인지
- 서비스 Root Directory가 저장소 루트인지
- Builder가 Dockerfile인지
- Start command가 `npm run start`인지
- Pre-deploy command가 `npm run db:deploy`인지
- Healthcheck path가 `/api/health`인지
- Variables에 `DATABASE_URL`, `NEXT_PUBLIC_APP_URL`, `HONZANG_ACCESS_CODE`, `HONZANG_ACCESS_TOKEN_SALT`가 있는지
