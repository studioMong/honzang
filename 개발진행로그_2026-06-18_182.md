## 작업 주제
- 홈택스/신고서에 옮길 핵심 금액을 한 표로 묶는 `신고서 입력값 요약` 추가

## 구현 내용
- 리포트 화면에 `신고서 입력값 요약` 패널을 추가했다.
- 요약 항목:
  - 부가세: 과세 매출 공급가액, 매출세액, 매입세액 공제 추정, 예상 납부/환급
  - 원천세: 원천세 후보 지급액, 예상 원천세
  - 법인세: 기간 손익, 승인 분개/원장, 재무제표 초안, 통장 잔액 대조
- 각 행에 `근거`, `상태`, `최종 확인`을 함께 표시해 사용자가 어떤 숫자를 어디서 대조해야 하는지 확인할 수 있게 했다.
- 리포트 스냅샷 payload에 `filingInputSummaryRows`를 추가했다.
- 저장된 리포트 상세 화면에서도 저장 당시 `신고서 입력값 요약`을 볼 수 있게 했다.
- 신고 패키지 JSON/ZIP/XLSX에 입력값 요약을 포함했다.
  - ZIP: `csv/filing-input-summary.csv`
  - XLSX: `입력값요약` 시트
- README MVP 범위와 production smoke 검증 문구를 갱신했다.
