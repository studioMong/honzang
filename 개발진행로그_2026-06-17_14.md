# 개발진행로그 14

## 목표

사용자가 장부 지식이 부족해도 어떤 신고 준비를 먼저 해야 하는지 리포트 화면에서 확인할 수 있게 한다.

## 변경 내용

- 리포트 화면 상단에 `신고 일정` 표를 추가했다.
- 증빙 정리, 부가세, 원천세, 법인세 순서로 준비 항목을 보여준다.
- 회사 설정의 과세유형, 결산월, 대표자 급여, 직원 급여, 외주 지급 여부를 일정 상태 계산에 사용한다.
- 신고 일정 CSV 다운로드를 추가했다.
- 리포트 스냅샷 저장 payload에 신고 일정 행을 포함하고, 저장된 리포트 상세에서도 일정 표를 볼 수 있게 했다.

## 일정 계산 기준

- 부가세: 선택 기간이 1월-6월이면 7월 25일, 7월-12월이면 다음 해 1월 25일을 기본 기한으로 표시한다.
- 원천세: 선택 기간의 지급월 다음 달 10일을 기본 기한으로 표시한다.
- 법인세: 회사 결산월 말일 이후 3개월째 말일을 기본 기한으로 표시한다.
- 증빙 정리: 선택 기간 말일을 월 마감 기준일로 표시한다.

## 검증 결과

- `npm run typecheck` 통과
- `npm run lint` 통과
- `npm run verify:samples` 통과
- `npm run build` 통과

## 제한 사항

- 세무 신고 기한은 법령 개정, 공휴일, 홈택스 공지에 따라 달라질 수 있으므로 앱에서는 기본 기한과 다음 작업 안내로만 제공한다.
- 공식 국세청/법령 출처 링크는 별도 문서화 단계에서 보강해야 한다.
