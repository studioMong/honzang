# 개발진행로그 54 - 대시보드 오늘 할 일 패널

## 진행 내용

- 대시보드에 `오늘 할 일` 패널을 추가했다.
- 현재 장부 상태를 보고 다음 처리 순서를 자동으로 뽑는다.
  - 법인 기본정보/과금 단가 설정
  - 거래 CSV 업로드
  - 계정과목 확정
  - 증빙 매칭
  - 자동분개 승인
  - 검토함 처리
  - 월별 리포트 저장
  - 월 마감 잠금
- 항목별 버튼을 눌러 해당 작업 화면으로 바로 이동할 수 있게 했다.
- 모바일 폭에서는 액션 버튼이 행 아래로 내려가도록 반응형 스타일을 추가했다.

## 검증

- `npm run typecheck` 통과
- `npm run lint` 통과
- `npm run build` 통과
- `npm run smoke:prod` 통과
- Browser 데스크톱 검증
  - 대시보드에서 `오늘 할 일` 패널 렌더링 확인
  - 샘플 데이터 기준 5개 우선순위 항목 렌더링 확인
  - 페이지 전체 가로 오버플로우 없음
  - 콘솔 오류 없음
- Browser 모바일 폭 390px 검증
  - `오늘 할 일` 패널 렌더링 확인
  - 액션 버튼이 항목 아래로 줄바꿈되는 구조 확인
  - 페이지 전체 가로 오버플로우 없음
  - 콘솔 오류 없음
- `첫 항목 열기` 버튼 클릭 검증
  - 샘플 데이터 기준 첫 항목인 법인 기본정보 입력 화면으로 이동 확인
  - 콘솔 오류 없음

## 참고

- 이 패널은 신고 절차 자체를 확정하는 기능이 아니라, 현재 입력된 장부 데이터를 기준으로 우선 처리할 항목을 정렬하는 안내 화면이다.
