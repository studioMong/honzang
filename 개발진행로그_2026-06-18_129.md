# 개발 진행 로그 129

## 작업 단위
- 백업 복원 전체교체에서 기존 커스텀 계정과목이 남는 문제를 보강했다.

## 변경 파일
- `src/app/api/backups/restore/route.ts`

## 구현 내용
- `clearCompanyData`가 거래, 증빙, 분개, 리포트, 마감, 거래처, 자동분류, CSV 템플릿, 활동 로그를 삭제한 뒤 기본 계정코드가 아닌 기존 계정과목도 삭제하도록 했다.
- 이후 `restoreAccounts`가 기본 계정과 백업 계정만 다시 upsert하므로, 복원 파일에 없는 과거 커스텀 계정과목이 전체교체 후 남지 않는다.

## 실행 명령
- `npm run typecheck`
- `npm run lint`
- `git diff --check`
- `npm run verify:mvp`
- `if [ -n "$DATABASE_URL" ]; then printf 'DATABASE_URL=set\n'; else printf 'DATABASE_URL=missing\n'; fi`
