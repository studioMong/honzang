# 개발진행로그 48 - 월 마감 잠금

## 진행 내용

- `ClosingPeriod` Prisma 모델과 마이그레이션을 추가했다.
- `/api/closing-periods`를 추가해 월별 마감 잠금, 마감 해제, 마감 목록 조회를 지원한다.
- 거래, CSV 가져오기, 증빙, 분개, 리포트 API에 마감 기간 쓰기 차단을 연결했다.
- 리포트 화면에서 월별 마감 잠금/해제 버튼과 잠금 상태를 표시하도록 연결했다.
- 워크스페이스 백업 JSON/ZIP과 백업 복원 dry-run 카운트에 월 마감 상태를 포함했다.

## 의도

- 신고자료가 확정된 월은 이후 수정으로 장부가 흔들리지 않게 잠근다.
- 잠금은 UI에서 먼저 안내하고, 서버 API에서 다시 차단해 직접 API 호출도 막는다.
- 백업/복원에도 마감 상태를 포함해 복구 후 장부 통제 상태가 유지되도록 한다.
